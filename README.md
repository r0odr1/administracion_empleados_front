# Employee Management App

Full Stack: .NET 10 (API REST) + Angular + SQLite + JWT

## Requisitos
- .NET 10 SDK
- Node.js 18+
- Angular CLI (`npm install -g @angular/cli`)

## Ejecutar el Backend
cd EmployeeApi
dotnet run
→ API disponible en https://localhost:5269

## Ejecutar el Frontend
cd employee-frontend
npm install
ng serve
→ App disponible en http://localhost:4200

## Credenciales de prueba
- Usuario: admin
- Contraseña: admin123

## Probar el CRUD
1. Ir a http://localhost:4200
2. Iniciar sesión con las credenciales anteriores
3. Crear, editar y eliminar empleados desde la interfaz
